HTML

Es un lenguaje informático que estructura todas las páginas web en internet.

El código HTML les dice a los navegadores web cómo mostrar contenido como imágenes, enlaces y texto en una página web.

Sus siglas en inglés significan HyperText Markup Language, lo que en español significa, Lenguaje de Marcado de Hipertexto.

Etiquetas

  • <button> es una etiqueta de apertura. Le dice al navegador web que queremos comenzar a mostrar un botón. Debe finalizar con una etiqueta de cierre. </button>
  • <img src="imagen" alt="Logo OscarDev"> es una etiqueta de apertura. Le dice al navegador web que queremos mostrar una imagen. Esta no posee una etiqueta de cierre y nos permite mostrar una imagen como la siguiente.
